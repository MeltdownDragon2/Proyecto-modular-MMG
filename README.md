## Overview
En este git se encuentran los codigos utilizados en el desarrollo del proyecto MECHANOBAND, los datos recabados
se encuentran disponibles en la plataforma Zenodo mediante el siguiente DOI: https://doi.org/10.5281/zenodo.19775977

## Contenido
Se encuentra aqui el codigo referente a la calibracion del equipo, adquicision de datos y evaluacion grupal
de estos, debido a la anonimidad, se han eliminado los nombres de los sujetos de prueba
En caso de errores, contactar a los autores originales

## Notas de uso
Codigo realizado en formato MATLAB, es posible que ciertas carpetas deban cambiarse de nombre para su correcto
funcionamiento
